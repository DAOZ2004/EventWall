# Descripcion:

**EventWall es una aplicación web desarrollada con Django para gestionar comunidades y eventos.
Permite a los usuarios crear comunidades, unirse a ellas, publicar eventos, administrarlos y visualizar actividades dentro de cada comunidad. El sistema incluye autenticación de usuarios, control de permisos, buscador inteligente de comunidades y administración interna de eventos.**

# Características principales:

**-Registro, login y administración de usuarios.**

**-Creación y administración de comunidades.**

**-Creación, edición y eliminación de eventos asociados a comunidades.**

**-Vista de detalle para comunidades y eventos.**

**-Buscador integrado para encontrar comunidades por nombre o descripción.**

**-Interfaz limpia, responsiva y fácil de usar.**
